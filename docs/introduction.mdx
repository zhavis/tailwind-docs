# Tailwind CSS: A Utility-First Framework

## What is Tailwind CSS?

**Tailwind CSS** is a highly customizable, low-level CSS framework that provides building blocks for rapidly building modern websites without ever leaving your HTML. Unlike traditional CSS frameworks like Bootstrap or Foundation that offer pre-designed components, Tailwind takes a **utility-first** approach.

## The Core Philosophy

Instead of writing custom CSS for every design element, Tailwind gives you a comprehensive set of single-purpose utility classes that you can combine to create any design directly in your markup:

```html
<!-- Traditional CSS approach -->
<div class="card">...</div>

<!-- Tailwind utility-first approach -->
<div class="bg-white rounded-lg shadow-md p-6 max-w-sm mx-auto">
  <h2 class="text-xl font-bold text-gray-800 mb-4">Card Title</h2>
  <p class="text-gray-600">Card content goes here...</p>
</div>
```

## How It Works

1. **Utility Classes**: Single-purpose classes like `p-4` (padding), `text-center`, `bg-blue-500`
2. **Responsive Design**: Prefix utilities with breakpoints (`sm:`, `md:`, `lg:`, `xl:`)
3. **State Variants**: Add prefixes for states (`hover:`, `focus:`, `active:`)
4. **Dark Mode**: Easy dark theme support with `dark:` prefix
5. **Component Extraction**: Use `@apply` to create component classes from utilities

## Getting Started

```bash
# Installation
npm install -D tailwindcss
npx tailwindcss init

# Basic usage in your CSS
@tailwind base;
@tailwind components;
@tailwind utilities;
```
