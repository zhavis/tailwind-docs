# Tailwind Basics

Tailwind CSS is built around the concept of **utility-first classes**, where each class applies a single CSS property. Understanding the core utilities is essential to building layouts quickly and efficiently.

---

## 1. Utility-First Concept

Instead of writing custom CSS for every element:

```css
.button {
  background-color: blue;
  color: white;
  padding: 0.5rem 1rem;
  border-radius: 0.25rem;
}
```
[Start utility first concept](https://docs-hubb.netlify.app/docs/utilities/start)

Tailwind allows you to apply styles directly using classes:

```html
<button class="bg-blue-500 text-white px-4 py-2 rounded">
  Click Me
</button>
```

Each class corresponds to **one property**:

- `bg-blue-500` → background color
- `text-white` → text color
- `px-4` → horizontal padding
- `py-2` → vertical padding
- `rounded` → border radius

---

## 2. Colors

Tailwind provides a **built-in color palette** that covers shades for backgrounds, text, borders, and more.

### Examples:

```html
<div class="bg-red-500 text-white p-4 mb-2">Red Background</div>
<div class="bg-green-200 text-green-800 p-4 mb-2">Light Green</div>
<div class="text-blue-600">Blue Text</div>
```

- `bg-*` → background color
- `text-*` → text color
- `border-*` → border color

You can customize the colors in `tailwind.config.js`.

---

## 3. Spacing (Margin & Padding)

Tailwind uses a **spacing scale** (`0, 1, 2, 3, 4…`) representing multiples of `0.25rem` by default.

### Margin and Padding

```html
<div class="m-4 p-6 bg-gray-200">Box with spacing</div>
```

- `m-4` → margin on all sides
- `mt-2` → margin-top
- `mb-6` → margin-bottom
- `px-4` → padding-left & padding-right
- `py-2` → padding-top & padding-bottom

### Negative Margin

```html
<div class="mt-[-10px]">Negative margin example</div>
```

---

## 4. Typography

Tailwind provides utilities for **font size, weight, line height, and text alignment**:

```html
<h1 class="text-4xl font-bold mb-4">Heading 1</h1>
<p class="text-base text-gray-700 leading-relaxed">
  This is a paragraph styled with Tailwind typography utilities.
</p>
```

- `text-*` → font size
- `font-bold`, `font-medium`, `font-light` → font weight
- `leading-*` → line height
- `text-center`, `text-left`, `text-right` → alignment

---

## 5. Borders and Rounded Corners

Add borders and radius with:

```html
<div class="border border-gray-300 rounded-lg p-4">
  Box with border and rounded corners
</div>
```

- `border` → adds border
- `border-gray-300` → sets border color
- `rounded` → default border radius
- `rounded-sm`, `rounded-md`, `rounded-lg`, `rounded-full` → custom radius

---

## 6. Shadows

Tailwind provides several shadow utilities:

```html
<div class="shadow-sm p-4 mb-2">Small shadow</div>
<div class="shadow-md p-4 mb-2">Medium shadow</div>
<div class="shadow-lg p-4">Large shadow</div>
```

- `shadow` → default shadow
- `shadow-sm`, `shadow-md`, `shadow-lg`, `shadow-xl`, `shadow-2xl` → different shadow sizes
- `shadow-none` → removes shadow

---

## 7. Widths, Heights, and Max Sizes

```html
<div class="w-64 h-32 bg-blue-200">Fixed width and height</div>
<div class="max-w-sm p-4 bg-gray-100">Max width example</div>
```

- `w-*` → width
- `h-*` → height
- `max-w-*`, `min-w-*` → max/min width
- `max-h-*`, `min-h-*` → max/min height

---

## 8. Display and Flex Utilities

Tailwind provides classes for **layout control**:

```html
<div class="flex items-center justify-between p-4 bg-gray-100">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

- `flex` → display flex
- `grid` → display grid
- `items-center` → vertical alignment
- `justify-between` → horizontal distribution

---

## 9. Example: Simple Card Component

```html
<div class="max-w-sm rounded overflow-hidden shadow-lg p-4 bg-white">
  <img class="w-full" src="https://via.placeholder.com/400x200" alt="Sample Image">
  <div class="px-4 py-2">
    <h2 class="font-bold text-xl mb-2">Card Title</h2>
    <p class="text-gray-700 text-base">This is a simple card component using Tailwind utilities.</p>
  </div>
</div>
```

This shows how **spacing, typography, borders, and shadows** can be applied with simple classes.

---

Tailwind basics cover **utilities for spacing, color, typography, layout, and shadows**. Mastering these will allow you to build most UI components without writing custom CSS.
