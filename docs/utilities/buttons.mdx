## Buttons 

Buttons let users interact with your site. Tailwind makes it easy to style them quickly.
### Tailwind Classes Cheat Sheet
So here i've just added the tailwind cheat sheet, which allows you to adjust the design of your buttons pretty easily : 

| Purpose       | Example Class                   |
| ------------- | ------------------------------- |
| Background    | `bg-blue-500`                   |
| Text Color    | `text-white`                    |
| Padding       | `py-2 px-4`                     |
| Border Radius | `rounded`                       |
| Hover Effect  | `hover:bg-blue-600`             |
| Disabled      | `opacity-50 cursor-not-allowed` |

### Basic Button styles
```html
<button class="bg-blue-500 text-white py-2 px-4 rounded">Primary</button>
<button class="bg-gray-200 text-gray-800 py-2 px-4 rounded">Secondary</button>
<button class="border border-gray-300 text-gray-700 py-2 px-4 rounded">Outlined</button>
```
### Changes The States
```html
<button class="bg-blue-500 hover:bg-blue-600 text-white py-2 px-4 rounded">Hover</button>
<button class="bg-blue-500 text-white py-2 px-4 rounded opacity-50 cursor-not-allowed" disabled>Disabled</button>
```

### Changes The Size
```html
<button class="text-sm py-1 px-3 rounded">Small</button>
<button class="py-2 px-4 rounded">Medium</button>
<button class="text-lg py-3 px-6 rounded-lg">Large</button>
```

### Button Variants
```html
<button class="bg-green-500 hover:bg-green-600 text-white py-2 px-4 rounded">Success</button>
<button class="bg-red-500 hover:bg-red-600 text-white py-2 px-4 rounded">Danger</button>
<button class="bg-yellow-500 hover:bg-yellow-600 text-white py-2 px-4 rounded">Warning</button>
```
### Buttons with Icons
```html
<button class="flex items-center bg-blue-500 text-white py-2 px-4 rounded">
  <svg class="w-4 h-4 mr-2" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 4v16m8-8H4"></path>
  </svg>
  Add Item
</button>
```
